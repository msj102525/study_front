이미지 처리 태그 연습 페이지

img 태그
웹 페이지에 그림 또는 사진(이미지 파일)을 표시할 때 사용하는 태그임
필수 속성은 src(source), src="이미지파일명.확장자"
웹에서 사용 가능한 이미지 파일의 종류 gif, jpg, png(웹 전용 압축포맷)
이미지파일의 폴더 위치 지정은 상대경로와 절대경로 둘 다 사용 가능함
상대경로 : 현재 문서를 기준으로 대상 파일을 찾아가는 경로임
절대경로 : 웹 어플리케이션의 root에서 시작하는 경로, "/context root명/디렉토리/파일명.확장명"
