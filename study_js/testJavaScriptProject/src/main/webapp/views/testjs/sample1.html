자바스크립트 작성 위치와 작성 방법

inline 방식 : 시작태그에 on이벤트이름 속성을 추가하고, 속성의 값에 자바스크립트 소스를 바로 작성하는 방식 on이벤트이름 속성은 자바스크립트가 제공하는 속성이며, 모든 태그에 사용할 수 있다.


Internal 방식 : 자바스크립트 소스를 script 태그 안에 함수(function) 단위로 작성해서 실행하는 방식, 기본 방식

script 태그의 작성 위치는 html 태그 밖, head 태그 안, body태그 안 어디에든 작성 가능함
script 태그가 문서 안에서 사용하는 횟수에도 제한 없음
script 태그 안에서 작성한 함수(function)를 태그에서 실행을 하려면, on이벤트명 속성을 추가하고 값에 실행항 함수명을 기술하면 됨, on이벤트명="실행할함수명(전달값)"


External 방식 : 별도의 자바스크립트 소스 파일(*.js)을 작성해 두고, 웹 문서에서 가져다 사용하는 방식

그림을 클릭하면 해당 img 태그의 src 속성 값을 바꾸는 toggleImage() 함수가 실행됨
그림을 클릭할 때마다 교대로 바뀜
